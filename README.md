# VIN Decoder

A React-based Single Page Application for decoding Vehicle Identification Numbers (VIN) using the NHTSA API.

## Features

- VIN input form with validation (length, characters)
- Displays decoding results in a table
- History of last 3 decoded VINs with re-load functionality
- Variables list page with details
- Responsive minimalistic design
- Error handling for API and validation

## Live Demo

[Deployed on Vercel: https://vin-decoder-omega.vercel.app/]

## Local Setup

1. Clone the repo
2. `npm install`
3. `npm run dev`

## Tech Stack

- React 18 + TypeScript
- Vite
- React Router DOM
- NHTSA Public API

## Deployment

Recommended: Vercel or Netlify for easy SPA deployment with client-side routing.
